# Pretendard 폰트 다운로드 안내

이 폴더에 Pretendard 폰트 파일을 추가해주세요.

## 다운로드 방법

1. [Pretendard GitHub Releases](https://github.com/orioncactus/pretendard/releases) 페이지로 이동
2. 최신 버전의 `Pretendard-1.3.9.zip` 파일을 다운로드
3. 압축을 해제한 후, `public/static` 폴더에서 다음 파일들을 복사:
   - `Pretendard-Regular.ttf` (또는 `.otf`)
   - `Pretendard-Medium.ttf` (또는 `.otf`)
   - `Pretendard-SemiBold.ttf` (또는 `.otf`)
   - `Pretendard-Bold.ttf` (또는 `.otf`)
4. 복사한 파일들을 이 폴더(`assets/fonts/`)에 넣어주세요

또는 [Pretendard CDN](https://cdn.jsdelivr.net/gh/orioncactus/pretendard@latest/packages/pretendard/dist/public/static/)에서 직접 다운로드할 수 있습니다.

## 필요한 파일 목록

- Pretendard-Regular.ttf (weight: 400)
- Pretendard-Medium.ttf (weight: 500)
- Pretendard-SemiBold.ttf (weight: 600)
- Pretendard-Bold.ttf (weight: 700)

폰트 파일을 추가한 후 `flutter pub get`을 실행해주세요.
